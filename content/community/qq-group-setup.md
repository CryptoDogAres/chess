# QQ Group Setup Guide for Xiangqi Channel

**Purpose:** Build a core community of xiangqi (Chinese chess) enthusiasts around the channel. QQ groups offer file sharing, announcements, and large group capacity -- ideal for an active chess community.

**Tasks completed:** cc1 (community channel setup), pb7 (platform community building)

---

## 1. Create the QQ Group

**Step-by-step:**

1. Open QQ on desktop or mobile
2. Click the "+" icon -> "Create Group"
3. Select category: "Game / Board Game" (or "Interest / Hobby")
4. Group name: **棋道象棋交流群**
   - Alternative names if branding differs: "棋道中国象棋", "[频道名]象棋交流群"
5. Set group avatar: Use channel logo or a clean xiangqi board image
6. Set group capacity: Start with 200-person group (free tier), upgrade to 500 or 1000 as needed

## 6. Admin Role Structure

| Role | Person | Responsibilities |
|------|--------|------------------|
| **群主 (Owner)** | Channel creator | Final decisions, group settings, content strategy |
| **管理员 A (Admin - Moderation)** | Trusted early member | Remove spam, enforce rules, approve join requests |
| **管理员 B (Admin - Activities)** | Active community member | Post weekly puzzles, organize discussions, track engagement |
| **管理员 C (Admin - Content)** | (Optional, add later) | Share FEN files, curate game records, maintain group files |

**When to add admins:**
- First admin at 50 members (moderation becomes necessary)
- Second admin at 150 members (activities need a dedicated person)
- Third admin at 300+ members (file management becomes a task)

**Admin selection criteria:**
- Active in group for 2+ weeks
- Helpful to other members
- Good knowledge of xiangqi
- Reliable and respectful

## 7. Weekly Activity Plan

### Monday: Weekly Puzzle (每周一题)

**Post format:**

```
【每周一题 第X期】

红先胜，请找出最佳着法：

[插入棋局图片]

FEN: [填入FEN字符串]

难度：★★☆☆☆（入门/中级/高级）

请在群内回复你的解法，周二公布答案！
提示：注意红车的位置。
```

- Source puzzles from your video content or classic endgame manuals
- Rotate difficulty: Week 1 easy, Week 2 medium, Week 3 hard, Week 4 easy
- Announce the answer Tuesday with a short explanation

### Wednesday: Discussion Day (名局讨论日)

**Post format:**

```
【名局讨论 第X期】

本周讨论：[棋手A] vs [棋手B]（[赛事名称]，[年份]）

视频链接：[填入对应视频链接]

讨论重点：
1. 第X步 [某个关键着法] 是否最优？
2. 黑方有没有更好的防守选择？
3. 这个开局体系你在实战中用过吗？

欢迎各抒己见，言之有理即可！
```

- Tie directly to that week's video content for cross-promotion
- Encourage members to watch the video first, then discuss

### Friday: Game Review Day (棋局复盘日)

**Post format:**

```
【复盘分享 第X期】

欢迎大家分享本周自己下过的棋局！

分享格式：
- 对弈平台：天天象棋 / 弈天 / 其他
- 执子：红/黑
- 开局：[开局名称]
- 结果：胜/负/和
- 关键转折：第X步
- 棋局截图或FEN

我会挑选1-2盘有代表性的棋局做点评！
```

- Select interesting submissions and provide brief analysis
- Best submissions could become future video content

### Weekend: Free Play (自由约战)

Members post "约战" to find opponents. Encourage use of online platforms (天天象棋, 弈天) and sharing results back in the group.

## 8. Group File Sharing Setup

Create these folders in the QQ group file section:

| Folder Name | Contents |
|-------------|----------|
| **新手资源包** | Beginner guides, piece movement rules, basic checkmates |
| **每周谜题** | Weekly puzzle images and FEN files, archived by date |
| **名局棋谱** | Game records (.pgn or .cbf) from analyzed famous games |
| **开局宝典** | Opening guides organized by system (中炮, 飞相, 仙人指路, etc.) |
| **群友对局** | Member-submitted games for review |
| **视频配套材料** | FEN positions and supplementary notes for channel videos |

**FEN sharing best practice:**

When sharing positions, always include:
1. A screenshot or diagram image
2. The FEN string for digital replay
3. A one-line description of the position

Example:
```
【仙人指路对局 关键局面】
FEN: rnbakabnr/9/1c5c1/p1p1p1p1p/9/9/P1P1P1P1P/1C5C1/9/RNBAKABNR w - - 0 1
说明：红方先手，开局第一步
```

## Growth Milestones

| Members | Action |
|---------|--------|
| 0-50 | Promote in every video, seed content yourself, be very active |
| 50-100 | Appoint first admin, start consistent weekly schedule |
| 100-200 | Add second admin, begin member-submitted content features |
| 200-500 | Upgrade group capacity, consider sub-groups by skill level |
| 500+ | Evaluate creating a QQ channel (频道) for more structured content |

The group should feel like a natural extension of your video content -- a place where the conversation continues after the video ends.
