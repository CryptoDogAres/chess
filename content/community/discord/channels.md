## 📋 INFO / 信息

### #welcome
- **Topic:** 欢迎来到棋道！请阅读规则。Welcome to ChessDAO! Please read the rules.
- **Permissions:** Everyone can read, only admins can post
- **Slowmode:** Off
- **Post:** The welcome message from `welcome-message.md`

### #announcements
- **Topic:** 新视频通知和重要公告 · New video alerts & important announcements
- **Permissions:** Everyone can read, only admins can post
- **Slowmode:** Off
- **Note:** Connect YouTube/Bilibili notification bot here

### #rules
- **Topic:** 服务器规则 · Server Rules — 违规将被警告或封禁 · Violations will result in warnings or bans
- **Permissions:** Everyone can read, only admins can post
- **Slowmode:** Off
- **Post the following rules:**

```
服务器规则 / Server Rules

1. 尊重所有成员 / Be respectful to all members
   - 禁止人身攻击、歧视或骚扰
   - No personal attacks, discrimination, or harassment

2. 使用正确的频道 / Use the correct channels
   - 请在相应频道发布内容
   - Post content in the appropriate channel

3. 禁止剧透谜题答案 / No puzzle spoilers
   - 不要在 #daily-puzzle 直接发答案，使用剧透标签 ||答案||
   - Don't post answers directly in #daily-puzzle, use spoiler tags ||answer||

4. 禁止垃圾信息 / No spam
   - 禁止重复发送、过度推广或无关链接
   - No repeated messages, excessive self-promotion, or irrelevant links

5. 禁止作弊讨论 / No cheating discussion
   - 禁止讨论或分享任何作弊工具或方法
   - Do not discuss or share cheating tools or methods

6. 内容必须适当 / Keep content appropriate
   - 禁止NSFW内容
   - No NSFW content

7. 听从管理员指示 / Follow moderator instructions
   - 管理员的决定是最终决定
   - Moderator decisions are final

违规处理 / Enforcement:
⚠️ 第一次：警告 / 1st offense: Warning
⚠️ 第二次：禁言24小时 / 2nd offense: 24h mute
⚠️ 第三次：封禁 / 3rd offense: Ban
```
