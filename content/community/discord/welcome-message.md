## 关于本服务器 / About This Server

**棋道 ChessDAO** 是一个象棋（中国象棋）学习与交流的社区。无论你是初学者还是高手，这里都有适合你的内容。

**棋道 ChessDAO** is a community for learning and discussing xiangqi (Chinese chess). Whether you're a beginner or an expert, there's something here for you.

我们通过每日谜题、对局分析、开局研究和社区锦标赛来一起提高棋力。

We improve together through daily puzzles, game analysis, opening study, and community tournaments.

## 服务器规则（简要）/ Rules (Summary)

1. **尊重他人** / Be respectful — 禁止攻击和骚扰 / No attacks or harassment
2. **正确使用频道** / Use correct channels — 内容请发到对应频道 / Post in the right place
3. **谜题不要剧透** / No puzzle spoilers — 用 `||答案||` 标签 / Use `||spoiler||` tags
4. **禁止垃圾信息** / No spam — 禁止刷屏和无关推广 / No flooding or irrelevant promotion
5. **禁止讨论作弊** / No cheating talk — 不分享作弊工具 / Don't share cheating tools
6. **听从管理员** / Follow moderators — 管理员决定为最终决定 / Mod decisions are final

完整规则请看 `#rules` / Full rules in `#rules`
