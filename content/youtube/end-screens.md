Create these playlists before uploading any videos:

| Playlist Name | Videos | Description |
|---------------|--------|-------------|
| Puzzle Shorts / 每日一题 | Videos 1-5 (ongoing) | Daily xiangqi puzzles — checkmate patterns, endgame challenges, tactical tests. New puzzle every day. |

## Subscribe Button Placement Rules

- **Long-form videos:** Always include subscribe button in end screen, positioned bottom-center
- **Shorts:** No end screen possible. Use verbal CTA at end of video ("Follow for more puzzles") and pinned comment
- **Channel trailer:** Large subscribe button, center position, appears at 0:50

## Cross-Promotion Matrix

This table shows which videos promote which other videos, ensuring traffic flows between content types:

| From Video | Promotes (End Screen) | Promotes (Cards) |
|------------|----------------------|-------------------|
| Puzzle Shorts (1-5) | N/A (pinned comments) | N/A |
| Famous Game (6) | Next Famous Game + Algorithm | Tutorial, Series pilot, Next episode |
| Rules Tutorial (7) | Pieces tutorial + First puzzle | Pieces tutorial, Cannon puzzle, Tutorial playlist |
| Pieces Tutorial (8) | Next tutorial + Famous Game | Rules recap, Knight puzzle, Cannon puzzle |
| Series Pilot (9) | Famous Game + Algorithm | First puzzle, Puzzle playlist, Famous Game |
| Trailer (10) | Rules Tutorial + Famous Game | None |

**Goal:** Every viewer path leads to either (a) the next video in a series, (b) a different content type to explore, or (c) the subscribe button.
