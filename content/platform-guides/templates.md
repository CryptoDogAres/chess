# Ready-to-Use Templates Collection

Copy-paste templates for video titles, descriptions, comments, CTAs, outreach messages, and more. Organized by platform and use case. Replace bracketed placeholders with your specific content.

## 1. Video Title Templates

### Douyin Titles (10 Templates)

Douyin titles should be under 55 characters, punchy, and curiosity-driven. Chinese language.

1. `这步棋99%的人下错了！你能看出正解吗？`
2. `[开局名称]最强陷阱，对手根本反应不过来`
3. `残局挑战：[棋子组合]杀法，你能几步解出？`
4. `象棋高手vs新手，差距到底在哪？`
5. `学会这招[战术名称]，棋力直接涨一级`
6. `[X]步绝杀！这道残局你敢挑战吗？`
7. `为什么你总是输在中局？这[X]个错误必须改`
8. `[大师名称]的经典名局，每一步都是教科书`
9. `象棋开局千万别走这步，不然直接崩盘`
10. `今日残局：[难度]级难度，评论区写出你的答案`

Bilibili titles can be longer (up to 80 characters). Informational and specific works best.

## 3. Pinned Comment Templates

Pinned comments boost engagement by prompting viewers to reply.

### Puzzle / Challenge Videos

**Chinese (Douyin / Bilibili):**
```
你觉得正确答案是什么？写在回复里！
答对的第一个人我会置顶你的评论 👀
```

**English (YouTube):**
```
What do you think is the best move here? Drop your answer below!
First correct answer gets pinned. Let's see who can solve it fastest.
```

### Tutorial / Educational Videos

**Chinese:**
```
这期讲的[主题]对你有帮助吗？
如果有不懂的地方，直接评论问我，每条都会回复！
```

**English:**
```
Was this helpful for your xiangqi study? Let me know in the replies.
If any part was unclear, ask below — I respond to every comment.
```

### Controversial / Opinion Videos

**Chinese:**
```
你同意我的分析吗？还是你有不同的看法？
评论区辩论一下，最有道理的回复我会置顶！
```

**English:**
```
Do you agree with my analysis? I know some players prefer a different approach here.
Share your thoughts — best counterargument gets pinned.
```

## 4. End-of-Video CTA Scripts

Spoken lines for the last 10–15 seconds of each video.

**CTA 1 — Standard:**
```
如果这期内容对你有帮助，别忘了点赞关注，我们下期再见！

```

觉得有用就点个赞，关注我，每天都有新残局等你挑战。
下期见！

```
如果你觉得今天的分析有价值，一键三连支持一下。
你们的三连是我更新最大的动力，感谢每一位支持我的朋友。
我们下期不见不散！

**CTA 1 — Standard:**
```
If you found this useful, hit the subscribe button and the notification bell
so you do not miss the next video. See you in the next one.
```

**CTA 2 — With engagement hook:**
```
What would you have played in that position? Let me know in the comments.

```

Subscribe and hit the bell so you do not miss it.
Check the playlist in the description for all previous episodes.

## 10. Audience Q&A / FAQ Response Templates

For recurring questions in comments or DMs.

**"How do I get better at xiangqi?"**

Chinese:
```
推荐三步走：
1. 每天做5道残局题（本频道残局系列可以跟）
2. 学1-2个开局体系，练到熟悉为止
3. 多下实战，每盘结束后复盘

坚持一个月你会看到明显进步！
```

English:
```
My recommended path:
1. Solve 5 endgame puzzles daily (check my puzzle playlist)
2. Learn 1-2 opening systems deeply before branching out
3. Play real games and review every one afterward

Stick with it for a month and you will see real improvement.
```

**"What software/app do you use?"**

```
录屏/Board display: [软件名，如天天象棋/XBoard/Xiangqi Studio]
剪辑/Editing: 剪映 (shorts) + Premiere Pro (long-form)
直播/Streaming: OBS Studio
封面/Thumbnails: Canva + Photoshop
```

**"Can you analyze my game?"**

Chinese:
```
可以的！把你的对局发到[QQ群号/邮箱]，注明你执红还是执黑。
我会在之后的视频里挑选粉丝对局进行分析。

English:
```
Yes! Send your game to [email] or post it in the comments (FEN or screenshot).
I regularly feature viewer games in my analysis videos.
```
